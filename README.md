# Syrah manuscript
Starting data and code for the _in silico_ experiments for *Syrah: an analysis pipeline to maximize spatial transcriptomic output*
The [main Syrah R package GitHub repostiory is here](https://github.com/0x644be25/Syrah) and contains a [Tutorial and Quick Start Guide]([https://github.com/0x644BE25/Syrah/blob/main/Syrah_tutorial.md](https://github.com/0x644BE25/Syrah/tree/main?tab=readme-ov-file#tutorialquick-start-guide)) to walk you through installing and running Syrah.
