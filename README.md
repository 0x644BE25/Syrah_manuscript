# *Syrah: a pipeline to maximize spatial transcriptomics data output*

Additional data and code for the analysis (including _in silico_ experiments) presented in *Syrah: an analysis pipeline to maximize spatial transcriptomic output*.


The [GitHub repostiory for the Syrah R package is here](https://github.com/0x644be25/Syrah) and contains a [Tutorial and Quick Start Guide]([https://github.com/0x644BE25/Syrah/blob/main/Syrah_tutorial.md](https://github.com/0x644BE25/Syrah/tree/main?tab=readme-ov-file#tutorialquick-start-guide)) to walk you through installing and running Syrah.
