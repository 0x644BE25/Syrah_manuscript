# *Syrah: a pipeline to maximize spatial transcriptomics data output *

Additional data and code for the analysis (including _in silico_ experiments) presented in _[Syrah: a pipeline to maximize spatial transcriptomics data output](http://academic.oup.com/g3journal/advance-article/doi/10.1093/g3journal/jkag107/8667091)_(Brewster et al., 2026)


The [GitHub repostiory for the Syrah R package is here](https://github.com/0x644be25/Syrah) and contains a [Tutorial and Quick Start Guide]([https://github.com/0x644BE25/Syrah/blob/main/Syrah_tutorial.md](https://github.com/0x644BE25/Syrah/tree/main?tab=readme-ov-file#tutorialquick-start-guide)) to walk you through installing and running Syrah.

# Troubleshooting
The main Syrah repo has more information, but if you're still having problems don't hesitate to contact me through [email](cbrewster@stowers.org) or on GitHub.
